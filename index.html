vue(21)-指令-自定义指令

前言

例如:v-text,v-for,v-html这些都是vue自带的指令,但是有时候也许自带指令不够时候我们需要自定义指令

内容

自定义指令运用类的方法:Vue.directive() 

Vue.directive(指令名称,function(){

});

调用指令名称:在指令名称前面加上’v-‘ 比如：red指令 则调用时候为:v-red

1.改变背景

1.指令无参 

<body>
<div id="box">
   <span v-red>
       {{msg}}
   </span>
</div>
<script>
Vue.directive("red",function(){
    this.el.style.background="red";
});
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     }
 });
</script>
</body>

2.指令传参 

<body>
<div id="box">
    <input type="text" v-model="a"><br>
   <span v-red="a">
       {{msg}}
   </span>
</div>
<script>
Vue.directive("red",function(val){
    this.el.style.background=val;
});
 new Vue({
     el:"#box",
     data:{
         a:"blue",
         msg:"hello world"

     }
 });
</script>
</body>

vue(17)-computed计算属性

前言

当一个数据发生变化时候,另一个数据也发生变化,比如下面数据:当改变a时候b的值会在a的值基础上加1

 new Vue({
    el:'#box',
    data:{
        msg:"hello"
        a:1,
        b:1
    }
});

内容

1.计算属性使用:computed
1.1 最基本的形式
computed:{//默认调用get
  b:function(){
  return 值;   
 }
}

computed里面是属性,它的值取决于return 后的结果

1.2.复杂形式(computed里面是个对象,b里面也是一个对象)
computed:{
  b:{
      get:function(){
         return this.a+2;
       },
      set:function(val){
         this.a = val;
       }
   }
}

计算属性其实最主要的用法是get,set方法,以上方法就是是get方法
假如执行如下方法

document.onclick=function(){
    vm.b=101;
}

其实就是调用set方法

1.3 对比

computed里面放置业务代码,一定记得return,data里面是基本数据

2.代码
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="./js/vue.js"></script>
</head>
<body>
<div id="box">
     a: {{a}}<br>
     b:{{b}}
</div>

<script>
var vm = new Vue({
    el:'#box',
    data:{
        msg:"hello",
        a:1

    },
    computed:{
        b:function () {
            //业务逻辑代码
            return this.a+1;
        }
    }
});
document.onclick=function(){
    vm.a=10;
};
</script>
</body>
</html>

vue(14)-vue生存周期

前言

1.当我们用vue做一个数据回显(也就是当打开或者刷新页面时候在vue方法里面就马上执行这个函数,而不是使用window.onload),我们需要用到vue的生命周期中的一些方法。
2.注意:在我们创建vue对象时候,如果不加

window.onload=function(){
     new Vue({
         el:'#box',
         data:{
             msg:"hello vue"
         }
     });
 }

会报错:

原因就是box在其后面定义,而new Vue()先执行所以找不到,我们也可以将new Vue()放在box后面

内容

1. vue生命周期

要想在vue中实现类似于window.onload效果,我们需要知道vue生命周期:

1.new Vue()其实就是创建一个实例,实例什么时候创建完毕是有标志的钩子函数之一:created
2.将变成正真的数据(vue中的data中的msg中的well)需要进行编译：beforeCompile,Compiled
3.需要把编译好的数据添加到dom树上(文本节点插入到文档中:类似于window.onload)：ready
4.对象也可以销毁:beforeDestroy和destroyed(要想让beforeDestroy和destroyed执行,我们需要绑定事件执行vue里面自带的方法)
5.以上是基于vue1.0,,vue2.0将以上周期函数改名了

1.1钩子函数

created -> 实例已经创建
boforeCompile ->编译之前
compiled ->编译之后
ready ->插入到文档中
beforeDestroy ->销毁之前
destroyed ->销毁之后

2.代码
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>vue生命周期</title>
<script src="vue.js"></script>

</head>
  <body>
  <div id="box">
   {{msg}}
  </div>

  <script>
  var vm = new Vue({
       el:'#box',
       data:{
           msg:"well"
       },
       created:function(){
           alert("对象已经创建!");
       },
       beforeCompile:function(){
           alert("编译之前!");
       },
       compiled:function(){
           alert("编译之后");
       },
       ready:function(){
           alert("插入到文档中");
       },
       beforeDestroy:function () {
           alert("销毁前");
       },
       destroyed:function () {
           alert("销毁后");
       }
   });

   //点击文档之后销毁对象,对象的销毁并不会在浏览器里面自动执行,需要事件触发
   document.onclick=function(){
       vm.$destroy();
   }
</script>
</body>
</html>

vue(13)-微博评论示例

前言

基于vue的微博评论案例

内容

1.功能要求

1.点击”提交评论”,我们把提交的数据扔到下面留言框.
2.页面刷新时候我们需要展示已经添加的一些数据

2.实现要求
1.点击”提交评论”,我们把提交的数据扔到下面留言框.

a.点击事件:v-on:事件 @事件
b.发送数据请求:this.$http.get()/post()/jsonp() this.$http()
c:按照接口请求数据 

接口为:weibo.do?act=add&content=xxx    添加一条
        返回：{error:0, id: 新添加内容的ID, time: 添加时间}
  this.$http({//默认为get
    url:url,
    data:{
      act:"abc",
      content://content值是textarea数据值,我们绑定v-model,并在data中定义
    },

  }).then(function(res){
    //返回时候渲染留言
     console.log(res.data);
  });

d:请求成功时候渲染:在vue的data里面定义msgData[{},{}],里面内容是返回结果
e:在view里面遍历用vue-for
f:暂无留言出来条件:v-show msgData.length==0
5:清空textarea

2.页面刷新时候我们需要展示已经添加的一些数据

a.获取某一页
b.接口:weibo.php?act=get&page=1 获取一页数据 

返回：[{id: ID, content: "内容", time: 时间戳, acc: 顶次数, ref: 踩次数}, {...}, ...]

c.交互:getPageData(n)
d:需要在页面一加载调用:能不能提供一个类似于：window.load函数,其实是vue有自带的:vue的生存周期
e:vue类似于window.load函数就是:ready

3.代码:
<!doctype html>

    window.onload=function(){
        new Vue({
            el:".znsArea",
            data:{
               msg:"abc",
                msgData:[/*{
                  content:"aaa",
                    time:121212,
                    acc:0,
                    ref:4,
                    id:12
                }*/]
            },
            methods:{
                add:function(){
                    //发送数据
                    this.$http({
                        url:URL,
                        data:{
                            act:"abc",
                            content:this.msg
                        }
                    }).then(function(res){
                        var json = res.data;
                        //添加返回结果到msgData
                        this.msgData.unshift({
                            content:this.t1,
                            time:json.time,
                            acc:0,
                            ref:0,
                            id:json.id
                        });
                        //清空textarea数据
                        this.t1="";
                    });
                },
                getPageData:function (n) {
                    this.$http({
                        url:URL,
                        data:{
                           act:"get",
                            page:n
                        }
                    }).then(function(res){
                       //将分页数据加载到数据中(msgData)
                        var arr = res.data;
                        //遍历数据
                        for(var i=0;i<arr.length;i++){
                            this.msgData.push({
                                content:arr[i].content,
                                time:arr[i].time,
                                acc:arr[i].acc,
                                ref:arr[i].ref,
                                id:arr[i].id
                            })
                        }
                    });
                }
            },
            ready:function(){
                //页面一加载就查询第一页
