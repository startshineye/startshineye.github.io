vue(25)-组件

前言

什么是组件：组件是Vue.js最强大的功能之一。组件可以扩展HTML元素，封装可重用的代码。在较高层面上，组件是自定义的元素，Vue.js的编译器为它添加特殊功能。在有些情况下，组件也可以是原生HTML元素的形式，以is特性扩展。组件就是一个大的对象

1.组件可以看成是HTML的扩展:比如：可以看成是HTML的扩展，但是在HTML中没有,我们需要自己来扩展,调用,这样就引出了组件。
2.定义一个组件:

Vue.component("aaa",Aaa);

2.（就是把extend出对象放入component中)

vue(24)-动画

前言

一般我们可以通过改变css来制造动画,当然也可以通过vue.js的自带来构建动画.

内容

vue ->过渡(动画)
本质走的是css3:transtion,animation

用法:
transition = “fade”:定义一个渐入渐出的动画

实例:点击按钮实现一个div的淡入淡出 

<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="./js/vue.js"></script>
<style>
    #div1{
        width: 200px;
        height: 200px;
        background: red;
    }

    <!--定义动画-->
        .fade-transition{
            transition: 1s all ease;
        }
    .fade-enter{
        opacity: 0;
    }
    .fade-leave{
        opacity: 0;
        transform: translateX(200px);
    }
</style>
</head>
<body>
<div id="box">
    <input type="button" value="按钮" @click="toggle">
    <div id="div1" v-show="bSign" transition="fade"></div>

    </div>
<script>
new Vue({
    el:'#box',
    data:{
        bSign:true
    },
    methods:{
        toggle:function(){
             this.bSign=!this.bSign;
        }
        /*toggle(){
            this.bSign=!this.bSign;
        }*/
    }
});
</script>
</body>

但是对于动画我们一般用其他的一个框架:animate.css
定义动画时候vue自带了一个属性：transitions 所有动画的调用都是基于此. 

<style>
    #box{
        width:400px;
        margin: 0 auto;
    }

        width:100px;

        bSign:true

            this.bSign=!this.bSign;

            enterClass:'zoomInLeft',
            leaveClass:'zoomOutRight'
        }
    }

</script>
</body>

vue(21)-指令-自定义指令

前言

例如:v-text,v-for,v-html这些都是vue自带的指令,但是有时候也许自带指令不够时候我们需要自定义指令

内容

自定义指令运用类的方法:Vue.directive() 

Vue.directive(指令名称,function(){

});

调用指令名称:在指令名称前面加上’v-‘ 比如：red指令 则调用时候为:v-red

1.改变背景

1.指令无参 

<body>
<div id="box">
   <span v-red>
       {{msg}}
   </span>
</div>
<script>
Vue.directive("red",function(){
    this.el.style.background="red";
});
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     }
 });
</script>
</body>

2.指令传参 

<body>
<div id="box">
    <input type="text" v-model="a"><br>
   <span v-red="a">
       {{msg}}
   </span>
</div>
<script>
Vue.directive("red",function(val){
    this.el.style.background=val;
});
 new Vue({
     el:"#box",
     data:{
         a:"blue",
         msg:"hello world"

     }
 });
</script>
</body>

2.拖拽
<body>
<div id="box">
    <div v-drag :style="{width:'100px',height:'100px',background:'yellow',position:'absolute',right:0,top:0}">

    </div>
</div>
<script>
Vue.directive("drag",function(){
    var oDiv = this.el;
    oDiv.onmousedown=function(ev){
        var disX = ev.clientX-oDiv.offsetLeft;
        var disY = ev.clientY-oDiv.offsetTop;

        document.onmousemove=function(ev){
            var l = ev.clientX-disX;
            var t = ev.clientY-disY;
            oDiv.style.left=l+'px';
            oDiv.style.top=t+'px';
        };
        document.onmouseup=function(){
            document.onmousemove=null;
            document.onmouseup=null;
        }
    }
});
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     }
 });
</script>
</body>
