1.解决ip被封问题
1.1 设置每个页面抓取时间间隔

  1. 设置每个页面抓取时间间隔,降低被封概率,之前我们已经采取了措施,就是设置抓取时间间隔设置成3秒,5秒,一般来说 抓取时间超过1秒才算合理

  2. 设置成3 5秒抓取时间比较固定,一般来说,网站能够分析出你抓取的频率,鉴定出是爬虫,我们设置随机数（1 2 3 4 5）秒 总是变化中的 让网站相信 我们是合法用户

  3. 我们在开始爬虫方法中读取配置文件的时间改成随机数:Math.random

  1. 缺点:此种方式在【网站监测到同一个ip访问过多情况下 也会封ip情况】 并不能解决,就是不让一天访问过多,

    1.2 购买代理ip库

java爬虫-15-多线程爬虫

前言

  1. 实现多线程爬虫

  2. 之前的爬虫程序,是单线程,爬取网页url，一次只能爬取一个,假如网页url比较多的话,爬虫时间就会很长,针对于单个应用,爬虫时间比较长的话,我们并行执行,时间就会缩短了。

  3. 假如单线程爬虫需要24分钟,如果我们开启两个线程,那么十多个小时就能够搞定
  4. 线程并不是越多越好,它受电脑资源的限制,电脑cpu所提供的资源有限,当资源到达瓶颈时候,其他线程将会排队

内容

1.定义线程池
  1. 1.新建resources下配置文件:

  1. 2.定义工具类:LoadPropertyUtil中方法:getConfig(String key)

  2. 3.在开始执行类中:StartDSJ 定义一个固定的线程池属性

    public void startSpider(){

       while(true){
    //从队列中提取需要解析的url
     //String url = urlQueue.poll();
     String url = repositoryService.poll();
     //判断url是否为空
     if(StringUtils.isNotBlank(url)){
         //执行多线程
         newFixedThreadPool.execute(new Runnable() {
             @Override
             public void run() {
                 System.out.println("当前第"+Thread.currentThread().getId()+"个线程");
                 //下载
                 Page page = StartDSJ.this.downloadPage(url);
                 //解析
                 StartDSJ.this.processPage(page);
                 List<String> urlList = page.getUrlList();

                 //循环抓取
                 for(String eachUrl:urlList){
                     //循环遍历时候,根据每一个url的前缀是否是 http://tv.youku.com/search/index
                     if(eachUrl.startsWith("http://tv.youku.com/search/index")){
                         StartDSJ.this.repositoryService.addHighLevel(eachUrl);
                     }else{
                         StartDSJ.this.repositoryService.addLowLevel(eachUrl);
                     }
                     // this.urlQueue.add(eachUrl);
                 }
                 if(page.getUrl().startsWith("http://www.youku.com/show_page")){
                     //存储数据
                     StartDSJ.this.processPage(page);
                 }

                 //为了防止频繁短时间抓取,ip被封锁,我们引入
                 ThreadUtil.sleep(Long.parseLong(LoadPropertyUtil.getConfig("millions_3")));
             }
         });
     }else{
         System.out.println("队列中的电视剧url解析完毕，请等待！");
         //线程睡眠5秒
         ThreadUtil.sleep(Long.parseLong(LoadPropertyUtil.getConfig("millions_5")));
     }
   }
}

      * 页面下载工具类

     public class PageDownLoadUtil {
    /**
     * 设置用户代理，模拟浏览器
     */
    private final static String USER_AGENT="Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/45.0.2454.101 Safari/537.36";
    /**
     * 获取页面内容
     * @param url
     * @return
     */
    public static String getPageContent(String url){
        //创建builder
        HttpClientBuilder builder = HttpClients.custom();

        //build一个CloseableHttpClient
        CloseableHttpClient client = builder.build();
        //创建一个Get请求
        HttpGet request = new HttpGet(url);

        //执行请求并返回
        String content = null;
        try {
            //设置用户代理-模拟浏览器
            request.setHeader("User-Agent",USER_AGENT);
            CloseableHttpResponse response = client.execute(request);
            HttpEntity entity = response.getEntity();

            //将entity转换成String
            content = EntityUtils.toString(entity);
        } catch (IOException e) {
            e.printStackTrace();
        }
        return content;
    }
}
