html-css(6)-css属性选择器

前言

属性:为html元素提供各种附加信息的就是属性,它总以“属性名=属性值”这种名值对的形式出现，而且属性总是在HTML元素的开始标签中进行定义。
例如:

这里的class就是属性。一个元素里可以有多个属性，各属性之间以空格相分离。

内容

1. [attribute]

当元素具备属性时候,不管其属性是否为空,或者其他

2. [attribute=”value”] attr等于value

精确只有value值的属性才会被渲染

3. [attribute~=”value”] attr包含value字符串(完整单词)

只需要完整包含于value值的属性将会被渲染.

4.[attribute*=”value”] attr 包含vaule字符串

只需要包含字符串

5.[attribute|=”value”] attr以vaule(完整单词,用-隔开)开始

只要包含就可以选中

6.[attribute^=”value”] attr以vaule开始

7.[attribute$=”value”] attr以vaule结束

FingerprintJS-浏览器端实现指纹识别

前言

在很多情况下我们需要识别浏览器,从而来确定唯一性,比如在线客服中访客的标识,一般我们是不能通过js获取电脑的mac地址的,为了确定访客的唯一性,也许可以根据cookie来实现,但是cookie对于清浏览器缓存的人来说,可能会被清除掉,从而使访客不准.进而引入了FingerprintJS

内容

FingerprintJS 是一个快速的浏览器指纹库，纯 JavaScript 实现，没有依赖关系。默认情况下，使用 Murmur Hash 算法返回一个32位整数。Hash 函数可以很容易地更换.

官方网站

1.什么是指纹识别

FingerPrint 即我们常说的指纹识别，使用手指和拇指前端的纹理按下的纹印来鉴定身份。指纹是鉴别身份的一种可靠的方法，因为每个人的每个指头上的纹理排列各不相同而且不因发育或年龄而改变。

　　什么是浏览器指纹？

其实这是 Electronic Frontier Foundation 研究出来的技术，可以匿名识别 Web 浏览器，准确率高达94%。通过查询浏览器的代理字符串，屏幕色深，语言，插件安装与支持的 MIME 类型，时区偏移量和其他功能，如本地存储和会话存储等等，然后这些值通过散列函数传递产生指纹，不需要通过 Cookie 存储就可以识别浏览器。

2.IE浏览器的识别

通过检测浏览器安装的插件来识别是浏览器指纹重要的特征，然后在 IE 浏览器中，不可能只去检测浏览器列表和它们的 MIME 类型，因此需要设置 ie_activex: true 选项，这样 FingerprintJS 将会检测一下插件：

  1. Adobe Reader
  2. Adobe Flash Player
  3. Apple QuickTime
  4. Real player (all versions)
  5. Shockwave player
  6. Windows Media Player
  7. Microsoft Silverlight
  8. Skype ClickToCall
