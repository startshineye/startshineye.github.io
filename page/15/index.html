vue(23)-监听数据变化

前言

之前我们经过a数据变化b也会跟着变化,使用的是计算属性:computed,我们可以通过监听数据变化实现:vm.$wactch();

内容

1.监听数据变化

vm.$watch(name,fnCb);其中name是Vue中data里面要变化属性，fnCb是回调函数 

<body>
<div id="box">
    <input type="text" v-model="a"><br>
    a:{{a}}<br>
    msg:{{msg}}

</div>
<script>
var vm = new Vue({
     el:"#box",
     data:{
       a:"a",
         b:'',
         msg:''
     },
 });
vm.$watch("a",function(){
    this.msg="a数据发生变化"+this.a;
});
</script>
</body>

vue(22)-自定义键盘信息

前言

在键盘按下一些事件中我们可以通过@keydown.按键 来实现键盘事件,但是有些事件,比如：ctrl事件是不能通过@keydown.ctrl来实现的,需要

内容

1.enter事件

<body>
<div id="box">
    <input type="text" @keydown.enter="show">
</div>
<script>
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     },
     methods:{
         show:function(){
             alert("enter");
         }
     }
 });
</script>
</body>

2.ctrl按键
 document.onkeydown=function(ev){
    alert(ev.keyCode);
};

得到ctrl的keyCode值为17

1.方式一: 

<input type="text" @keydown.17="show">

2.方式二:
Vue.directive(“on”).keyCodes.ctrl=17; 

<body>
<div id="box">
    <input type="text" @keydown.ctrl="show">
</div>
<script>
 Vue.directive("on").keyCodes.ctrl=17;
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     },
     methods:{
         show:function(){
             alert("ctrl");
         }
     }
 });
</script>
</body>

vue(21)-指令-自定义指令

前言

例如:v-text,v-for,v-html这些都是vue自带的指令,但是有时候也许自带指令不够时候我们需要自定义指令

内容

自定义指令运用类的方法:Vue.directive() 

Vue.directive(指令名称,function(){

});

调用指令名称:在指令名称前面加上’v-‘ 比如：red指令 则调用时候为:v-red

1.改变背景

1.指令无参 

<body>
<div id="box">
   <span v-red>
       {{msg}}
   </span>
</div>
<script>
Vue.directive("red",function(){
    this.el.style.background="red";
});
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     }
 });
</script>
</body>

2.指令传参 

<body>
<div id="box">
    <input type="text" v-model="a"><br>
   <span v-red="a">
       {{msg}}
   </span>
</div>
<script>
Vue.directive("red",function(val){
    this.el.style.background=val;
});
 new Vue({
     el:"#box",
     data:{
         a:"blue",
         msg:"hello world"

     }
 });
</script>
</body>

2.拖拽
<body>
<div id="box">
    <div v-drag :style="{width:'100px',height:'100px',background:'yellow',position:'absolute',right:0,top:0}">

    </div>
</div>
<script>
Vue.directive("drag",function(){
    var oDiv = this.el;
    oDiv.onmousedown=function(ev){
        var disX = ev.clientX-oDiv.offsetLeft;
        var disY = ev.clientY-oDiv.offsetTop;

        document.onmousemove=function(ev){
            var l = ev.clientX-disX;
            var t = ev.clientY-disY;
            oDiv.style.left=l+'px';
            oDiv.style.top=t+'px';
        };
        document.onmouseup=function(){
            document.onmousemove=null;
            document.onmouseup=null;
        }
    }
});
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     }
 });
</script>
</body>

vue(20)-过滤器

前言

之前我们接触过简单的过滤器,比如:capitalize uppercase

内容

1.vue常见基本过滤器

capitalize -> 首字母大写
uppercas -> 变大写
debounce(需要配合事件) time :延迟 

<!DOCTYPE html>

</head>
<body>
<div id="box">
    <input type="button" value="show" @click="show | debounce 2000">
</div>

<script>
var vm = new Vue({
    data:{
    },
    methods:{
        show:function(){
            alert(1);
        }
    }
}).$mount("#box");

</body>
</html>

延迟2秒执行

2.配合数组使用的过滤器

比如我们展示一个数组时候,里面有100条数据,但是我们只需要展示前50条,这个时候用limitBy

2.1 limitBy

1.limitBy ->限制几个
2.limitBy 参数1 参数2 ->从哪开始取几个 

<div id="box">
    取2个:
   <ul>
       <li v-for="items in arr | limitBy 2">
             {{items}}
       </li>
   </ul>
    取2个从倒数第三取:
    <ul>
        <li v-for="items in arr | limitBy 2 arr.length-3">
            {{items}}
        </li>
    </ul>
</div>
<script>
var vm = new Vue({
    data:{
        arr:[1,2,3,4,5,6]
    },
    methods:{

    }
}).$mount("#box");
</script>

2.2 filterBy过滤

过滤出包含”o”的数据 

body>
<div id="box">
   按照输入的字母来过滤数组:

    <br>
    <ul>
        <li v-for="items in arr | filterBy a">
              {{items}}
        </li>
    </ul>
</div>
<script>
var vm = new Vue({

2.3 orderBy排序

有时候我们希望数据过来之后排序,使用orderby

3.自定义过滤器

对于时间过滤器,vue没有自带的过滤器使用,我们需要自定义过滤器

自定义过滤器:
Vue是一个类,通过类上的一个方法filter来自定义过滤器 

Vue.filter(name,function(parameter,a,b){
  //回调函数
});
parameter是要过滤的数据 a,b是调用时过滤器后面的参数
3.1 自定义时间过滤器
<body>

    var oData = new Date(val);
    return oData.getFullYear()+"-"+(oData.getMonth()+1)+"-"+oData.getDate()+" "+oData.getHours()+":"+oData.getMinutes()+":"+oData.getSeconds();
});
var vm = new Vue({
    data:{
      date:Date.now()
    },
    methods:{

    }

</script>

vue(18)-vue实例简单方法

前言

var vm = new Vue();创建一个对象,由此实例可以获取一些简单有用的方法

内容

统一观点:Vue实例所带的方法和属性带有:”$”

1.vue实例简单方法

1.vm.$el ->获取元素 类似于document.getElementById(“id”) 

vm.$el.style.background = 'red'  //元素变红

2.vm.$data ->就是data 

vm.$data

3.vm.$mount(“#box”) -> 手动挂载vue程序

var vm = new Vue(
   data:{
    a:1  
   }
).$mount("#box");

var vm = new Vue({
   aa:11,
   show:function(){

   },
   data:{
     a:1
   }

5.vm.$destroy ->销毁对象

6.vm.$log -> 查看现在数据的状态

vue(17)-computed计算属性

前言

当一个数据发生变化时候,另一个数据也发生变化,比如下面数据:当改变a时候b的值会在a的值基础上加1

 new Vue({
    el:'#box',
    data:{
        msg:"hello"
        a:1,
        b:1
    }
});

内容

1.计算属性使用:computed
1.1 最基本的形式
computed:{//默认调用get
  b:function(){
  return 值;   
 }
}

computed里面是属性,它的值取决于return 后的结果

1.2.复杂形式(computed里面是个对象,b里面也是一个对象)
computed:{
  b:{
      get:function(){
         return this.a+2;
       },
      set:function(val){
         this.a = val;
       }
   }
}

计算属性其实最主要的用法是get,set方法,以上方法就是是get方法
假如执行如下方法

document.onclick=function(){
    vm.b=101;
}

其实就是调用set方法

1.3 对比

computed里面放置业务代码,一定记得return,data里面是基本数据

2.代码
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="./js/vue.js"></script>
</head>
<body>
<div id="box">
     a: {{a}}<br>
     b:{{b}}
</div>

<script>
var vm = new Vue({
    el:'#box',
    data:{
        msg:"hello",
        a:1

    },
    computed:{
        b:function () {
            //业务逻辑代码
            return this.a+1;
        }
    }
});
document.onclick=function(){
    vm.a=10;
};
</script>
</body>
</html>

vue(14)-vue生存周期

前言

1.当我们用vue做一个数据回显(也就是当打开或者刷新页面时候在vue方法里面就马上执行这个函数,而不是使用window.onload),我们需要用到vue的生命周期中的一些方法。
2.注意:在我们创建vue对象时候,如果不加

window.onload=function(){
     new Vue({
         el:'#box',
         data:{
             msg:"hello vue"
         }
     });
 }

会报错:

原因就是box在其后面定义,而new Vue()先执行所以找不到,我们也可以将new Vue()放在box后面

内容

1. vue生命周期

要想在vue中实现类似于window.onload效果,我们需要知道vue生命周期:

1.new Vue()其实就是创建一个实例,实例什么时候创建完毕是有标志的钩子函数之一:created
2.将变成正真的数据(vue中的data中的msg中的well)需要进行编译：beforeCompile,Compiled
3.需要把编译好的数据添加到dom树上(文本节点插入到文档中:类似于window.onload)：ready
4.对象也可以销毁:beforeDestroy和destroyed(要想让beforeDestroy和destroyed执行,我们需要绑定事件执行vue里面自带的方法)
5.以上是基于vue1.0,,vue2.0将以上周期函数改名了

1.1钩子函数

created -> 实例已经创建
boforeCompile ->编译之前
compiled ->编译之后
ready ->插入到文档中
beforeDestroy ->销毁之前
destroyed ->销毁之后

2.代码
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>vue生命周期</title>
<script src="vue.js"></script>

</head>
  <body>
  <div id="box">
   {{msg}}
  </div>

  <script>
  var vm = new Vue({
       el:'#box',
       data:{
           msg:"well"
       },
       created:function(){
           alert("对象已经创建!");
       },
       beforeCompile:function(){
           alert("编译之前!");
       },
       compiled:function(){
           alert("编译之后");
       },
       ready:function(){
           alert("插入到文档中");
       },
       beforeDestroy:function () {
           alert("销毁前");
       },
       destroyed:function () {
           alert("销毁后");
       }
   });

   //点击文档之后销毁对象,对象的销毁并不会在浏览器里面自动执行,需要事件触发
   document.onclick=function(){
       vm.$destroy();
   }
</script>
</body>
</html>
