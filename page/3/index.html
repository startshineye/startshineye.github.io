vue(10)-模板

前言

Vue.js 使用了基于 HTML 的模版语法，允许开发者声明式地将 DOM 绑定至底层 Vue 实例的数据。
Vue.js 的核心是一个允许你采用简洁的模板语法来声明式的将数据渲染进 DOM 的系统。
结合响应系统，在应用状态改变时， Vue 能够智能地计算出重新渲染组件的最小代价并应用到 DOM 操作上。

内容

两个大括号模板 数据更新模板变化

1.代码

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="vue.js"></script>
<script>
    window.onload=function(){
        new Vue({
            el:"#box",
            data:{
                msg:'abc'
            }
        })
    }
</script>
</head>
<body>
<div id="box">
   <input type="text" v-model="msg"><br>
   {{msg}}<br>
</div>
</body>
</html>

vue(8)-事件

前言

vue类似于js也是有自己相应的事件,并且有自己的简写形式：v-on:click/mouseover….这些事件都有自己的简写形式

内容

1.事件

v-on:事件 来表示vue里面的事件
简写:
@click/mouseover…. 推荐
v-on: 等价于 @

1.v-on:实例 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="vue.js"></script>
<script>
   window.onload=function(){
       var vm = new Vue({
           el:"#box",
           data:{

           },
           methods:{
               show:function () {
                   alert(1);
               }
           }
       });
   }
</script>
</head>
<body> <div id="box">
       <input type="button" value="按钮" v-on:click="show()">
       <input type="button" value="等价按钮" @click="show()">
  </div>
</body>
</html>
2.事件对象

@click=”show($event)”代表点击事件的事件对象 从$event中我们可以拿到很多点击事件的东西

2.1 冒泡事件

1.事件冒泡:当我们点击一个事件时候会弹出其他父事件:如下:弹出1再弹出2 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="vue.js"></script>
<script>
   window.onload=function(){
       var vm = new Vue({
           el:"#box",
           data:{
           },
           methods:{
               show:function () {
                   alert(1);
               },
               show2:function(){
                   alert(2);
               }
           }
       });
   }
</script>
</head>
<body> <div id="box">
       <div @click="show2();">
           <input type="button" value="按钮" @click="show()">
       </div>
  </div>
</body>
</html>

2.阻止冒泡: a)event.cancelBubble=true; b)@click.stop 推荐

a)event.cancelBubble=true;

           },

                   alert(1);//
                   ev.cancelBubble=true;
               },

                   alert(2);
               }
           }

默认行为(默认事件):contextmenu右击时候不仅弹出alert还弹出说明
阻止默认行为: a):event.preventDefault(); b):@contextmenu.prevent 推荐
