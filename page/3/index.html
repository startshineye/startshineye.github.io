java获取图片大小高宽

java获取图片

在web开发过程中,我们需要上传图片,比如电商上就运用了大量图片,为了提高访问效率,我们需要对图片进行验证.以下就是java图片验证的基础.

java获取图片的代码

import java.awt.image.BufferedImage;  
import java.io.File;  
import java.io.FileInputStream;  
import java.io.FileNotFoundException;  
import java.io.IOException;  
import javax.imageio.ImageIO;  
public class PictureCheck {  
public static void main(String[] args) throws FileNotFoundException, IOException {  
    // 通过流 从C:/Users/yexinming/Downloads/目录下读取图片路径  
    File file = new File("C:/Users/yexinming/Downloads/2.jpg");  
    System.out.println(file);// C:/Users/yexinming/Downloads/2.jpg  
    // Java将一副图片加载到内存中的方法是：  
    BufferedImage image = ImageIO.read(new FileInputStream(file));  
    // image可以获得图片的详细信息  
    int width = image.getWidth();// 获取图片宽度  
    int height = image.getHeight();// 获取图片高度  
    System.out.println(file.length() / 1024.0);// 获取文件大小kb  
 }
}

Markdown入门

#Markdown–快速入门

前言:

Markdown是一种可以使用普通文本编辑器编写的标记语言,它可以使普通文本内容具有一定的格式.
目前被越来越多的写作爱好者,程序员,撰稿者使用.其语法简单,常用字符不超过十个,相比于HTML(HyperText Markup Language)来说,十分轻量,学习成本低.

Markdown概述

Markdown 是一种用来写作的轻量级「标记语言」，它用简洁的语法代替排版，而不像一般我们用的字处理软件 Word 或 Pages 有大量的排版、字体设置。它使我们专心于码字，用「标记」语法，来代替常见的排版格式.

Markdown优点

超级文本标记语言文档制作不是很复杂，但功能强大，支持不同数据格式的文件镶入,其优点如下:
一:专注于文本内容而不是排版
二:兼容所有文本编辑器和字体软件
三:能导出HTML,pdf,md文件
四:易学,上手快

##Markdown编辑工具
Markdown编辑工具很多,以下是不同操作系统下常用的工具
.Mac:Mou
.Linux:Atom
.Windows:MarkdownPad

Markdown简要语法

标题

文章需要使用标题时,只需要在这段文字前加上”#”,Markdown支持六种标题,一般#后面加上一个空格.

列表

如同 HTML中列表,在Markdown下,列表分为:有序列表和无序列表.
有序列表加1. 2. 3. …
无序列表加-或者*

图片与链接

图片插入和链接插入语法差别只在一个!上

引用

引用别人一小段别处的句子,只需要在文本前加入>即可

粗体与斜体

粗体:用两个包含一段文本
斜体:用一个包含一段文本

表格

表格相对来说比较复杂
例子:
| Tables | Are | Cool |
| ————- |:————-:| —–:|
| col 3 is | right-aligned | $1600 |
| col 2 is | centered | $12 |
| zebra stripes | are neat | $1 |
