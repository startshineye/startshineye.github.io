vue(24)-动画

前言

一般我们可以通过改变css来制造动画,当然也可以通过vue.js的自带来构建动画.

内容

vue ->过渡(动画)
本质走的是css3:transtion,animation

用法:
transition = “fade”:定义一个渐入渐出的动画

实例:点击按钮实现一个div的淡入淡出 

<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="./js/vue.js"></script>
<style>
    #div1{
        width: 200px;
        height: 200px;
        background: red;
    }

    <!--定义动画-->
        .fade-transition{
            transition: 1s all ease;
        }
    .fade-enter{
        opacity: 0;
    }
    .fade-leave{
        opacity: 0;
        transform: translateX(200px);
    }
</style>
</head>
<body>
<div id="box">
    <input type="button" value="按钮" @click="toggle">
    <div id="div1" v-show="bSign" transition="fade"></div>

    </div>
<script>
new Vue({
    el:'#box',
    data:{
        bSign:true
    },
    methods:{
        toggle:function(){
             this.bSign=!this.bSign;
        }
        /*toggle(){
            this.bSign=!this.bSign;
        }*/
    }
});
</script>
</body>

但是对于动画我们一般用其他的一个框架:animate.css
定义动画时候vue自带了一个属性：transitions 所有动画的调用都是基于此. 

<style>
    #box{
        width:400px;
        margin: 0 auto;
    }
    #div1{
        width:100px;
        height:100px;
        background: red;
    }
</style>
</head>
<body>
<div id="box">
<input type="button" value="按钮" @click="toggle">
<div id="div1" class="animated" v-show="bSign" transition="bounce"></div>
</div>

<script>
new Vue({
    el:'#box',
    data:{
        bSign:true
    },
    methods:{
        toggle:function () {
            this.bSign=!this.bSign;
        }
    },
    transitions:{ //定义所有动画名称
        bounce:{
            enterClass:'zoomInLeft',
            leaveClass:'zoomOutRight'
        }
    }
});
</script>
</body>

vue(22)-自定义键盘信息

前言

在键盘按下一些事件中我们可以通过@keydown.按键 来实现键盘事件,但是有些事件,比如：ctrl事件是不能通过@keydown.ctrl来实现的,需要

内容

1.enter事件

<body>
<div id="box">
    <input type="text" @keydown.enter="show">
</div>
<script>
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     },
     methods:{
         show:function(){
             alert("enter");
         }
     }
 });
</script>
</body>

2.ctrl按键
 document.onkeydown=function(ev){
    alert(ev.keyCode);
};

得到ctrl的keyCode值为17

1.方式一: 

<input type="text" @keydown.17="show">

2.方式二:
Vue.directive(“on”).keyCodes.ctrl=17; 

<body>
<div id="box">
    <input type="text" @keydown.ctrl="show">
</div>
<script>
 Vue.directive("on").keyCodes.ctrl=17;
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     },
     methods:{
         show:function(){
             alert("ctrl");
         }
     }
 });
</script>
</body>

vue(21)-指令-自定义指令

前言

例如:v-text,v-for,v-html这些都是vue自带的指令,但是有时候也许自带指令不够时候我们需要自定义指令

内容

自定义指令运用类的方法:Vue.directive() 

Vue.directive(指令名称,function(){

});

调用指令名称:在指令名称前面加上’v-‘ 比如：red指令 则调用时候为:v-red

1.改变背景

1.指令无参 

<body>
<div id="box">
   <span v-red>
       {{msg}}
   </span>
</div>
<script>
Vue.directive("red",function(){
    this.el.style.background="red";
});
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     }
 });
</script>
</body>

2.指令传参 

<body>
<div id="box">
    <input type="text" v-model="a"><br>
   <span v-red="a">
       {{msg}}
   </span>
</div>
<script>
Vue.directive("red",function(val){
    this.el.style.background=val;
});
 new Vue({
     el:"#box",
     data:{
         a:"blue",
         msg:"hello world"

     }
 });
</script>
</body>

2.拖拽
<body>
<div id="box">
    <div v-drag :style="{width:'100px',height:'100px',background:'yellow',position:'absolute',right:0,top:0}">

    </div>
</div>
<script>
Vue.directive("drag",function(){
    var oDiv = this.el;
    oDiv.onmousedown=function(ev){
        var disX = ev.clientX-oDiv.offsetLeft;
        var disY = ev.clientY-oDiv.offsetTop;

        document.onmousemove=function(ev){
            var l = ev.clientX-disX;
            var t = ev.clientY-disY;
            oDiv.style.left=l+'px';
            oDiv.style.top=t+'px';
        };
        document.onmouseup=function(){
            document.onmousemove=null;
            document.onmouseup=null;
        }
    }
});
 new Vue({
     el:"#box",
     data:{
         msg:"hello"
     }
 });
</script>
</body>

vue(20)-过滤器

前言

之前我们接触过简单的过滤器,比如:capitalize uppercase

内容

1.vue常见基本过滤器

capitalize -> 首字母大写
uppercas -> 变大写
debounce(需要配合事件) time :延迟 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="./js/vue.js"></script>
</head>
<body>
<div id="box">
    <input type="button" value="show" @click="show | debounce 2000">
</div>

<script>
var vm = new Vue({
    data:{
    },
    methods:{
        show:function(){
            alert(1);
        }
    }
}).$mount("#box");
</script>
</body>
</html>

延迟2秒执行

2.配合数组使用的过滤器

比如我们展示一个数组时候,里面有100条数据,但是我们只需要展示前50条,这个时候用limitBy

2.1 limitBy

1.limitBy ->限制几个
2.limitBy 参数1 参数2 ->从哪开始取几个 

<div id="box">
    取2个:
   <ul>
       <li v-for="items in arr | limitBy 2">
             {{items}}
       </li>
   </ul>
    取2个从倒数第三取:
    <ul>
        <li v-for="items in arr | limitBy 2 arr.length-3">
            {{items}}
        </li>
    </ul>
</div>
<script>
var vm = new Vue({
    data:{
        arr:[1,2,3,4,5,6]
    },
    methods:{

    }
}).$mount("#box");
</script>

2.2 filterBy过滤

过滤出包含”o”的数据 

body>
<div id="box">
   按照输入的字母来过滤数组:
    <input type="text" v-model="a">
    <br>
    <ul>
        <li v-for="items in arr | filterBy a">
              {{items}}
        </li>
    </ul>
</div>
<script>
var vm = new Vue({
    data:{
        a:'',
        arr:["apple","orange","banana","tomato"]
    },
    methods:{

    }
}).$mount("#box");
</script>
</body>

2.3 orderBy排序

有时候我们希望数据过来之后排序,使用orderby
orderBy就是正序倒序排列:orderBy 1/-1 ->正序排列/倒序排列 

<body>
<div id="box">
   按照输入的字母来排序数组:
    <input type="text" v-model="a">
    <br>
    <ul>
        <li v-for="items in arr | orderBy a">
              {{items}}
        </li>
    </ul>
</div>
<script>
var vm = new Vue({
    data:{
        a:'',
        arr:["apple","orange","banana","tomato"]
    },

    }

3.自定义过滤器

对于时间过滤器,vue没有自带的过滤器使用,我们需要自定义过滤器

自定义过滤器:
Vue是一个类,通过类上的一个方法filter来自定义过滤器 

Vue.filter(name,function(parameter,a,b){
  //回调函数
});
parameter是要过滤的数据 a,b是调用时过滤器后面的参数
3.1 自定义时间过滤器
<body>
<div id="box">
   {{date | dateFormat}}
</div>
<script>
Vue.filter("dateFormat",function(val){
    var oData = new Date(val);
    return oData.getFullYear()+"-"+(oData.getMonth()+1)+"-"+oData.getDate()+" "+oData.getHours()+":"+oData.getMinutes()+":"+oData.getSeconds();
});
var vm = new Vue({
    data:{
      date:Date.now()
    },
    methods:{

    }
}).$mount("#box");
</script>
</body>

vue(17)-computed计算属性

前言

当一个数据发生变化时候,另一个数据也发生变化,比如下面数据:当改变a时候b的值会在a的值基础上加1

 new Vue({
    el:'#box',
    data:{
        msg:"hello"
        a:1,
        b:1
    }
});

内容

1.计算属性使用:computed
1.1 最基本的形式
computed:{//默认调用get
  b:function(){
  return 值;   
 }
}

computed里面是属性,它的值取决于return 后的结果

1.2.复杂形式(computed里面是个对象,b里面也是一个对象)
computed:{
  b:{
      get:function(){
         return this.a+2;
       },
      set:function(val){
         this.a = val;
       }
   }
}

计算属性其实最主要的用法是get,set方法,以上方法就是是get方法
假如执行如下方法

document.onclick=function(){
    vm.b=101;
}

其实就是调用set方法

1.3 对比

computed里面放置业务代码,一定记得return,data里面是基本数据

2.代码
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="./js/vue.js"></script>
</head>
<body>
<div id="box">
     a: {{a}}<br>
     b:{{b}}
</div>

<script>
var vm = new Vue({
    el:'#box',
    data:{
        msg:"hello",
        a:1

    },
    computed:{
        b:function () {
            //业务逻辑代码
            return this.a+1;
        }
    }
});
document.onclick=function(){
    vm.a=10;
};
</script>
</body>
</html>

vue(15)-防止刚加载时闪烁

前言

当网络慢时候，我们以

{{}}

加载vue数据时候,会出现首先显示两大括号,此时就需要用到vue中标签

内容

1.v-clock属性

v-clock[属性] 防止闪烁 用在比较大的段落 

 <style>
    [v-clock]{
       display: none;
    }
</style>
<div class="reply" v-for="item in msgData" v-cloak>
            <p class="replyContent">{{item.content}}</p>
            <p class="operation">
                <span class="replyTime">{{item.time}}</span>
                <span class="handle">
                    <a href="javascript:;" class="top">{{item.acc}}</a>
                    <a href="javascript:;" class="down_icon">{{item.ref}}</a>
                    <a href="javascript:;" class="cut">删除</a>
                </span>
            </p>
        </div>
2.v-text

以下实例:

<span>{{msg}}</span>

当网络慢的时候会出现双大括号显示,但是我们加载v-clock到上面就会比较浪费 

<span>{{msg}}</span>

对于小段落,我们可以使用v-text 

<span v-text="msg"></span>

以上永远不会出现双大括号,因为本来就不存在双大括号
