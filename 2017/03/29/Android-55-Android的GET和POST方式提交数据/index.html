Android(55)-Android的GET和POST方式提交数据

前言

  1. 1.Android中以GET和POST中去服务器取数据是一个常用的操作
  2. 2.服务器一般不是我们所关心的,android只是从别人已经搭建好的服务器取数据
  3. 3.get方式和post方式区别
    get:组拼url地址把数据拼接到url上 有大小限制1kb 4kb
    post:以请求体的形式把数据提交到服务器
  4. 4.耗时的(如：联网等)操作一定要放到子线程中间
  5. 5.展示数据更改ui不能在子线程中操作,需要在主线程中操作,可以使用runOnUiThread中进行,runOnUiThread是执行在主线程的,是被google封装了的方法。
  6. 6.本例子需要用土司提示,所以展示数据可以封装一下：showToast(String content)
  7. 7.联网操作一定要加入联网权限permission

内容

总体设计成四个控件：用户名,密码,get提交,post提交
