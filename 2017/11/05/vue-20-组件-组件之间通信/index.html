前言

  1. 组件间通信是组件开发时非常重要的一环，我们既希望组件的独立性，数据能互不干涉,又不可避免组件间会有联系和交互。
  2. Vue.js 在组件间通信这一部分既提供了:直接访问，自定义事件机制，通过广播、派发、监听等形式进行跨组件的函数调用。
