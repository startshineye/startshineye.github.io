vue(28)-vue-router

前言

  1. vue是MVVP框架,vue是SPA应用(单页面应用)
  2. vue-router:路由:根据不同url地址,出现不同效果
  3. 使用vue-router需要下载:vue-router.js
  4. 路由是基于:0.7.13

内容

1.单应用页面为:主页 home 新闻页 news
2.对于属性href在vue时候是不可用了,vue-router提供了v-link
3.v-link={path:url}
4.点击v-link所在标签时候,跳转显示的页面用
5.转转的连接：v-link 展示内容
6.必须要一个根组件:以上home和news本身也是一个组件

1.简单入门

需求：准备两个组件 主页 新闻 点击主页 新闻时候跳转到对应的主页和新闻的view(单页面应用)

view: 

<div id="box">
<!--点击部分 -->
<ul>
    <li>
        <a v-link="{path:'/home'}">主页</a>
    </li>
    <li>
        <a v-link="{path:'/news'}">新闻</a>
    </li>
</ul>

<!--展示视图-->
<div>
    <router-view></router-view>
</div>
</div>

model:

//1.准备根组件
   var App = Vue.extend();
 //2.Home,News组件
 var Home = Vue.extend({
     template:"<h3>我是主页</h3>"
 });
 var  News = Vue.extend({
     template:"<h3>我是新闻</h3>"
 });
 //3.准备路由
 var router = new VueRouter();
 //4.关联
 router.map({
     'home':{
         component:Home
     },
     'news':{
         component:News
     }
 });
 //5.启动路由
 router.start(App,'#box');//绑定到哪个根路由(App)的哪个容器(box)

2.跳转

当我们一打开上面的文件时候,页面初始化时并没有跳转(页面没有任何跳转)

跳转:router.redirect({
‘/‘:’/home’,
‘/aaa’:’/news’
});

3.路由嵌套(多级)
3.1 主页嵌套

需求:实现以下需求

1.为了简洁,我们将’home’变成模板
2.我们点击模板之后,会加入一个class

加入样式: 

<style>
    .v-link-active{
        font-size: 20px;
        color: #985f0d;
    }
</style>

3.在模板下加入

4.在home下面的模板下的路由的路径需要是:”/home/login”,”/home/reg” 

<template id="home">
    <h3>我是主页</h3>
    <div>
        <a v-link="{path:'/home/login'}">登录</a>
        <a v-link="{path:'/home/reg'}">注册</a>
    </div>
    <div>
        <router-view></router-view>
    </div>
</template>

5.由于登录注册的路由都是基于home的所以路由关联也是写在相应的下面:subRoutes 

router.map({
        'home':{
            component:Home,
            subRoutes:{
                'login':{
                    component:{
                        template:'<strong>我是登录信息</strong>'
                    }
                },
                'reg':{
                    component:{
                        template:'<strong>我是注册信息</strong>'
                    }
                }
            }
        }
    });

代码:

<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="../js/vue.js"></script>

<body>
<div id="box">

    </li>
    <li>
        <a v-link="{path:'/news'}">新闻</a>
    </li>
</ul>

<!--展示视图-->
<div>
    <router-view></router-view>
</div>
</div>
<template id="home">
<h3>我是主页</h3>
<!--其下路由应该在其模板之下-->
<div>
    <a v-link="{path:'/home/login'}">登录</a>
    <a v-link="{path:'/home/reg'}">注册</a>
</div>
<!--显示其下的路由-->
<div>
    <router-view></router-view>
</div>

<template id="news">
  <h3>我是新闻</h3>

<script>
  //1.准备根组件
    var App = Vue.extend();
  //2.Home,News组件
  var Home = Vue.extend({
      template:"#home"
  });
  var  News = Vue.extend({
      template:"#news"
  });
  //3.准备路由
  var router = new VueRouter();
  //4.关联
  router.map({
      'home':{
          component:Home,
          subRoutes:{
              'login':{
                  component:{
                      template:"<h3>我是登录</h3>"
                  }
              },
              'reg':{
                  component:{
                      template:'<h3>我是注册</h3>'
                  }
              }
          }
      },
      'news':{
          component:News
      }
  });
  //5.启动路由
  router.start(App,'#box');//绑定到哪个根路由(App)的哪个容器(box)

  //6.跳转
  router.redirect({
      '/':'home'
  })
</script>
</html>
