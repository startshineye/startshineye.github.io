Guava(1)-java.util.Collections-ArrayListMultimap

前言

  1. Guava工程包含了若干被Google的 Java项目广泛依赖 的核心库，例如：集合 [collections] 、缓存 [caching] 、原生类型支持 [primitives support] 、并发库 [concurrency libraries] 、通用注解 [common annotations] 、字符串处理 [string processing] 、I/O 等等。 所有这些工具每天都在被Google的工程师应用在产品服务中。

  2. API地址为:http://tool.oschina.net/apidocs/apidoc?api=guava

内容

1.概述

  1. This multimap allows duplicate key-value pairs. After adding a new key-value pair equal to an existing key-value pair, the ArrayListMultimap will contain entries for both the new value and the old value.(相同key可以存储多个值,不会覆盖)

  2. When iterating through the collections supplied by this class, the ordering of values for a given key agrees with the order in which the values were added.(遍历时候,顺序是和放入时候顺序一致)

3.This class is not threadsafe when any concurrent operations update the multimap. Concurrent read operations will work correctly. To allow concurrent update operations, wrap your multimap with a call to Multimaps.synchronizedListMultimap(com.google.common.collect.ListMultimap).(线程非安全的)

2.类继承关系

3.常用方法
1.create 用于创建一个一键多值的multimap集合
ArrayListMultimap<String, String> multimap = ArrayListMultimap.create();
2.put方法 往multimap集合中put元素
multimap.put("A","123");
multimap.put("A", "456");
multimap.put("B", "789");
multimap.put("C", "012");

输出:{A=[123, 456], B=[789], C=[012]}

3.size 获取集合大小
int size = multimap.size();

输出:4

4.get 由键获取值
List<String> list = multimap.get("A");

输出:[123, 456]

5.keys方法和keySet方法
 //keys获取multimap集合中的所有键，可以重复
// keySet获取multimap集合中的不重复键
 Multiset<String> keys = multimap.keys();
 Set<String> keySet = multimap.keySet();

输出:[A x 2, B, C]
输出:[A, B, C]
