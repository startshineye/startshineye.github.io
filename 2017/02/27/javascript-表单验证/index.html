javascript-表单验证

需求

  1. 用户名不能为空
  2. 用户名长度必须在[6-14]位之间
  3. 密码和确认密码必须一致
  4. 用户填写一个验证一个（不要全部填写完之后再验证），验证失败时候，一定要在后面提示错误消息
  5. 用户表单信息验证失败，阻止表单的提交

步骤

  1. 设置span标签，将来错误信息显示在span标签中，用户名后面有span，确认密码后面有span
  2. 填写一个验证一个，这里需要使用到失去焦点事件：blur，句柄：onblur，谁失去焦点?文本框/密码框
  3. 编写验证用户名的函数：checkUsername(username)，给该函数使用this.value的方式传参
  4. 获取焦点之后，清除错误消息，需要使用一个事件focus，句柄：onfocus
  5. 验证密码和确认密码是否一致
  6. 清除密码错误提示信息
  7. 表单验证全部通过再提交表单，只要有一项不合法，阻止表单提交
