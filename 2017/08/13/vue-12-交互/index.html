vue(12)-交互

前言

vue本身框架是不支持交互的,如果vue交互需要通过引入库:vue-resource

内容

  1. 引入vue-resource后，可以基于全局的Vue对象使用http，也可以基于某个Vue实例使用http。

  2. // 基于全局Vue对象使用http
    Vue.http.get(‘/someUrl’, [options]).then(successCallback, errorCallback);
    Vue.http.post(‘/someUrl’, [body], [options]).then(successCallback, errorCallback);

  3. // 在一个Vue实例内使用$http
    this.$http.get(‘/someUrl’, [options]).then(successCallback, errorCallback);
    this.$http.post(‘/someUrl’, [body], [options]).then(successCallback, errorCallback);

  4. 在发送请求后，使用then方法来处理响应结果，then方法有两个参数，第一个参数是响应成功时的回调函数，第二个参数是响应失败时的回调函数。

  5. then方法的回调函数也有两种写法，第一种是传统的函数写法，第二种是更为简洁的ES 6的Lambda写法：

1.传统写法 

this.$http.get('/someUrl', [options]).then(function(response){
// 响应成功回调
}, function(response){
// 响应错误回调
});

2.Lambda写法 

this.$http.get('/someUrl', [options]).then((response) => {
// 响应成功回调
}, (response) => {
// 响应错误回调
});

交互无非几种方式,如下:
1.get
2.post
3.jsonp

1.get
1.1 get请求普通文本

在html同级目录下新建a.txt 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="vue.js"></script>
<script src="vue-resource.js"></script>
<script>
    window.onload=function(){
        new Vue({
            el:'body',
            data:{

            },
            methods:{
                get:function(){
                    // 在一个Vue实例内使用$http
                    this.$http.get(
                        "a.txt"
                    ).then(function(response){
                        //输出状态码
                        var status = response.status;
                        alert(status);
                        //内容
                        var data = response.data;
                        alert(data);
                    },function(response){
                        alert(response);
                    });
                }
            }
        });
    }
</script>
</head>
<body>
<!--点击按钮获取a.txt里面内容-->
<input type="button" value="get" @click="get()">
</body>
</html>

2.post

post的时候,需要加上一个头,post请求时候需要加入一个参数:emulateJSON:true 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="vue.js"></script>
<script src="vue-resource.js"></script>
<script>
    window.onload=function(){
        new Vue({
            el:'body',
            data:{

            },
            methods:{
                post:function(){
                    // 在一个Vue实例内使用$http
                    this.$http.post(
                        "post.php",{
                            a:1,
                            b:2
                        },{
                            //post请求时候需要添加
                            emulateJSON:true
                        }
                    ).then(function(response){
                        //输出状态码
                        var status = response.status;
                        alert(status);
                        //内容
                        var data = response.data;
                        alert(data);
                    },function(response){
                        alert(response);
                    });
                }
            }
        });
    }
</script>
</head>
<body>
<!--点击按钮获取a.txt里面内容-->
<input type="button" value="post" @click="post()">
</body>
</html>
3.jsonp

1.由于jsonp是走的get请求,所以不需要加:emulateJSON:true
2.jsonp默认的返回参数是callback,如果需要更改：jsonp:”xx”

3.1 请求360数据

我们想获取本域之外的数据时候:比如获取360数据:
1.捕获接口:
https://sug.so.360.cn/suggest?callback=suggest_so&word=a

2.代码 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="vue.js"></script>
<script src="vue-resource.js"></script>
<script>
    window.onload=function(){
        new Vue({
            el:'body',
            data:{

            },
            methods:{
                jsonp:function(){
                    // 在一个Vue实例内使用$http
                    //连接:https://sug.so.360.cn/suggest?callback=suggest_so&word=a
                    this.$http.jsonp(
                        "https://sug.so.360.cn/suggest",{
                            word:"a"
                        },{
                            //post请求时候需要添加
                            emulateJSON:true
                        }
                    ).then(function(response){
                        //输出状态码
                        var status = response.status;
                        alert(status);
                        //内容
                        alert(response.data.s);
                    },function(response){
                        alert(response);
                    });
                }
            }
        });
    }
</script>
</head>
<body>
<!--请求360时候:默认jsonp是callback-->
<input type="button" value="jsonp" @click="jsonp()">
</body>
</html>
