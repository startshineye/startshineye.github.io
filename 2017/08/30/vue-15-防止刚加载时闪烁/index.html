vue(15)-防止刚加载时闪烁

前言

当网络慢时候，我们以

{{}}

加载vue数据时候,会出现首先显示两大括号,此时就需要用到vue中标签

内容

1.v-clock属性

v-clock[属性] 防止闪烁 用在比较大的段落 

 <style>
    [v-clock]{
       display: none;
    }
</style>
<div class="reply" v-for="item in msgData" v-cloak>
            <p class="replyContent">{{item.content}}</p>
            <p class="operation">
                <span class="replyTime">{{item.time}}</span>
                <span class="handle">
                    <a href="javascript:;" class="top">{{item.acc}}</a>
                    <a href="javascript:;" class="down_icon">{{item.ref}}</a>
                    <a href="javascript:;" class="cut">删除</a>
                </span>
            </p>
        </div>
2.v-text

以下实例:

<span>{{msg}}</span>

当网络慢的时候会出现双大括号显示,但是我们加载v-clock到上面就会比较浪费 

<span>{{msg}}</span>

对于小段落,我们可以使用v-text 

<span v-text="msg"></span>

以上永远不会出现双大括号,因为本来就不存在双大括号 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<script src="./js/vue.js"></script>
</head>
<body>
<div id="box">
    <span>{{msg}}</span><br>
    <span v-text="msg"></span>
</div>

<script>
new Vue({
    el:'#box',
    data:{
        msg:"hello"
    }
});
</script>
</body>
</html>
