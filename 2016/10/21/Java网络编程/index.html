Sock通信编程

1.socket
  1. 两个java应用程序可以通过一个双向的网络连接实现数据交换.这个双向链路的一端称为一个(socket相当于通信两端的插座)
  2. socket通常用来实现client-server连接
  3. java.net包中定义的两个类Socket和ServerSocket,分别用来实现双向连接的client和server端
  4. 建立连接时所需要的寻址信息为远程计算机的IP地址和端口号(Port number)
    Socket socket = new Socket(“192.168.10.100”,55555);
  5. TCP端口UDP端口分开的
    每一个机器端口号为两个字节,含有65536个(一个端口可以分配给一个应用程序:比如QQ).
