java反射-Class类

反射的基石-Class类

1.java类：

用于描述一类事物共性，该事物有什么属性和没什么属性，是由这个类实例对象确定

2.Class类：

java程序中各个java类，他们是否属于同一类事物，是不是可以用一个类描述这类事物呢？描述这类java类的类就是Class类

3.Class类描述了哪些方面的信息呢？

类的名字，类的访问属性，类属性的包名，字段，方法

4.区别：
  1. 众多的人用一个什么类表示？众多的java类用什么类表示？
    人：Person
    java类：Class
  2. Person类代表人，实例对象就是张三，李四这样一个个具体的人。Class类代表的java类，他的各个实例对象又是什么呢？
    对象各个类在内存中的字节码，例如：Person类的字节码，ArrayList类的字节码，等等。
  3. 创建对象
    Person p1 = new Person();
    Person p2 = new Person();

Class c1 = new Class()//错误的做法，没有这个构造方法，或者这个构造方法私有的

Class c2 = 字节码2
java对象的创建是首先将java类编译成字节码。class文件放到硬盘上，然后将硬盘上的二进制代码加载到内存中，用这个字节码复制出一个个对象。比如代码中运用到了date，math，person类，那么内存中就有三份字节码，三份字节码就是三个Class的实例对象。Class c1 = Person.class//字节码
Person.class代表Person类的字节码，类型是Class
p1.getClass()也可以得到Person的字节码
Class.forName(“全类名”);也是得到类的字节码

5.怎样生成字节码
  1. 类名.class 例如：Person.class
  2. 对象.getClass(),例如：new Date().getClass()
  3. Class.forName(“类名”),例如：Class.forName(“java.util.Date”)

  5. 字节码已经生成，但是虚拟机中还没有字节码，用类加载器加载字节码，并用Class.forName(“类名”)返回字节码
